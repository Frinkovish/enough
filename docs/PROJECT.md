# Enough

Enough is a mobile application helping people reduce addictions through a 20-minute delay technique.

When a craving happens, the user starts a 20-minute session.

During the session the app suggests meaningful actions:
- small tasks
- monthly goals
- physical activities
- spiritual activities
- productivity actions

After 20 minutes the user decides:
- smoke
- don't smoke

The app does not punish relapses.

The objective is progress, not perfection.

Core principle:

Every craving becomes progress.
